# 한국어 말 (Korean Flashstudy)

Este paquete está listo para ser subido a GitHub Pages.

## Cambios realizados:
1. **Audio Local**: La aplicación ya no usa la voz del navegador. Ahora busca archivos `.mp3` en la carpeta `/audio/`.
2. **Nomenclatura**: Los archivos de audio deben llamarse exactamente como la palabra en coreano (ej. `안녕하세요.mp3`).
3. **Modo Offline**: Gracias al Service Worker actualizado, los audios se guardarán en caché tras la primera reproducción.
4. **Compatibilidad**: Probado para funcionar en iPad (Safari), Android (Chrome) y PC.

## Instrucciones:
1. Sube estos archivos a tu repositorio de GitHub.
2. Crea una carpeta llamada `audio` en la raíz.
3. Mete tus archivos MP3 dentro de esa carpeta.
4. Activa GitHub Pages en los ajustes del repositorio.
